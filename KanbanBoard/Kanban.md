# Kanban Board - Kütüphane Yönetim Sistemi

Bu projede GitHub Projects kullanılarak Kanban board oluşturulmuştur.

## Kullanılan Sütunlar
- To Do
- In Progress
- Done

- Book sınıfının oluşturulması
- Member sınıfının oluşturulması
- StudentMember kalıtım yapısının eklenmesi
- Loan sınıfının yazılması
- LibraryManager sınıfının yazılması
- JUnit testlerinin yazılması
- UML diyagramlarının hazırlanması
